## Reverse Mode Automatic Differentiation Engine

This project is a course assigment for the Neural Network course.

The goal is to implement a simple reverse mode automatic differentiation engine.


## Files
`tensor.py` : Tensor implementation

`function.py` : Mathematical operations and graph functions

`module.py` : Neural network module structure

`optimizer.py` : Optimization utilities

`example.py` : Example usage

`result.html` : Results
